Detailed Description

Header file for the G_AccelHarmonic function.

Computes the gradient of the Earth's harmonic gravity field.

This function computes the gradient of the gravitational acceleration vector (∂a/∂r), derived from a spherical harmonics expansion of the Earth's gravity field, in the true-of-date reference system.
