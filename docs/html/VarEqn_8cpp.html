Detailed Description

Header file for the VarEqn function. Computes the time derivative of the combined state vector and state transition matrix for an Earth-orbiting satellite. The state transition matrix describes the sensitivity of the state with respect to initial conditions, which is fundamental in orbit determination and estimation theory.
