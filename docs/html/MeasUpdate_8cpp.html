Detailed Description

Source file for the MeasUpdate function implementation.

Performs a Kalman measurement update for the state vector and its covariance.

This function computes the Kalman gain, updates the state estimate based on the measurement residual, and adjusts the covariance matrix accordingly. The measurement noise is assumed to be diagonal.
